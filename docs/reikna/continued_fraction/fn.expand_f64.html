pub fn expand_f64(fraction: &ContinuedFraction) -> f64

Expand the continued fraction fraction one time, storing the result as an f64.

This is a helper function that calls expand_f64_ntimes() with n = 1. See the documentation for expand_f64_ntimes() for more information.

Panics

Panics if expand_f64_ntimes() panics.

Examples

use reikna::continued_fraction::expand_f64;
println!("[2, 1] = {}", expand_f64(&vec![2, 1]));
println!("[3] = {}", expand_f64(&vec![3]));

Outputs:

[2; 1] = 2.5
[3] = 3
